Połączenie PHP z różnymi bazami danych

PHP - MySQL

Przykład połączenia PHP z bazą danych MySQL:


        <?php
        $servername = "localhost";
        $username = "root";
        $password = "password";
        $dbname = "my_database";

        $conn = new mysqli($servername, $username, $password, $dbname);

        if ($conn->connect_error) {
            die("Połączenie nieudane: " . $conn->connect_error);
        }
        echo "Połączenie udane";
        $conn->close();
        ?>

PHP - JSON

Przykład odczytu danych z pliku JSON w PHP:


        <?php
        $json = file_get_contents("data.json");
        $data = json_decode($json, true);
        echo "Dane z pliku JSON: ";
        print_r($data);
        ?>

PHP - PostgreSQL

Przykład połączenia PHP z bazą danych PostgreSQL:


        <?php
        $host = "localhost";
        $port = "5432";
        $dbname = "my_database";
        $user = "postgres";
        $password = "password";

        $conn = pg_connect("host=$host port=$port dbname=$dbname user=$user password=$password");

        if (!$conn) {
            die("Połączenie nieudane.");
        }
        echo "Połączenie udane";
        pg_close($conn);
        ?>

PHP - MS SQL Server

Przykład połączenia PHP z bazą danych MS SQL Server Express:


        <?php
        $serverName = "localhost\\SQLEXPRESS";
        $connectionInfo = array("Database"=>"my_database", "UID"=>"user", "PWD"=>"password");
        $conn = sqlsrv_connect($serverName, $connectionInfo);

        if (!$conn) {
            die("Połączenie nieudane.");
        }
        echo "Połączenie udane";
        sqlsrv_close($conn);
        ?>

PHP - ODBC

Przykład połączenia PHP z bazą danych przy użyciu ODBC:


        <?php
        $dsn = "DSN=myDSN";
        $user = "user";
        $password = "password";

        $conn = odbc_connect($dsn, $user, $password);

        if (!$conn) {
            die("Połączenie nieudane.");
        }
        echo "Połączenie udane";
        odbc_close($conn);
        ?>
